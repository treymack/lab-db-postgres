# AGENTS.md

This file provides guidance to AI agents (Claude Code, Codex, etc.) when working with code in this repository.

## Overview

This is a lab repository for PostgreSQL experiments. It is currently in early/empty state — add experiments, scripts, or projects as needed.

## Purpose

Exploratory PostgreSQL work: schema design, query optimization, replication, extensions, benchmarking, etc. Intended to be run in VSCode.

## Agent Rules

### Git

- Do not include `Co-Authored-By` lines in commit messages.

### Markdown

- Always add a blank line after headings and around lists (MD022, MD032).
- All markdown files must end with a single trailing newline (MD047).
